# Roadmap

High-level direction for this guidelines pack. For detailed feature state, see [`FEATURES.md`](FEATURES.md).

## Near Term

- Expand convention files to cover additional languages and frameworks (TypeScript/JavaScript, Python applications, Go, Rust, Java).

## Later

- Mobile and frontend conventions.
- Optional provider-specific CI/CD workflow examples.

## No Plans

- Runtime, build output, or deployable application — this repository is a guidelines pack only.
